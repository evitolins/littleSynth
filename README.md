littleSynth
===============================================================================
A set of simple classes to create sounds and music using the HTML5 WebAudio
API.

Pure javascript, with no dependencies.  Only requires a modern browser.


Classes
-------
__Oscillator__

- setFreq (freq)
- setType (type)
- play ()
- stop ()
- pulse (start, length)
